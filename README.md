# Создание онлайн-платформы для поддержки молодежных инициатив, проведения мероприятий и развития ИТ-коммьюнити на базе молодежного центра

[![Python](https://img.shields.io/badge/Python-3.13+-lightblue.svg)](https://python.org)
[![Django](https://img.shields.io/badge/Django-5.2-092E20.svg)](https://djangoproject.com)
[![PostgreSQL](https://img.shields.io/badge/PostgreSQL-17-336791.svg)](https://postgresql.org)

Веб-платформа для молодёжного центра «База», предназначенная систематизации деятельности молодёжного центра, обеспечивающей информирование аудитории о мероприятиях, поддержку молодёжных инициатив и удобное взаимодействие пользователей с ресурсами центра..

## О проекте

Платформа решает проблему децентрализации цифровых сервисов молодёжного центра и предоставляет единое пространство для взаимодействия участников и организаторов.

**Ключевые возможности:**
- 📅 Публикация и просмотр мероприятий и новостей
- 👥 Управление клубами по интересам
- 💡 Подача заявок на молодёжные инициативы
- 👤 Личный кабинет с историей заявок
- 🔐 Разграничение прав доступа (гость / пользователь / администратор)
- ⚙️ Административная панель для управления контентом и модерации заявок

## Технологический стек

### Бэкенд
- **Python 3.13+** — язык программирования для реализа серверной логики
- **Django 5.2(включая ORM)** — веб-фреймворк для реализации серверной логики и объектно-реляционного отображения данных
- **PostgreSQL 17** — реляционная база данных для надежного хранения информации
- **Django Admin** — встроенная панель администратора для управления контентом

### Фронтенд
- **HTML5** — для построения структуры страниц
- **CSS3** — для стилизации и адаптивной верстки
- **JavaScript** — для реализации интерактивных элементов
- **AJAX** — для асинхронного обновления контента без перезагрузки страницы
- **Bootstrap** — для построения адаптивной сетки и ускорения разработки

### Безопасность
- CSRF-токены — защита от межсайтовых атак
- Валидация форм на стороне клиента и сервера
- Защита от SQL-инъекций (встроенная в Django ORM)
