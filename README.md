# Онлайн-платформа для поддержки молодёжных инициатив

[![Python](https://img.shields.io/badge/Python-3.13+-lightblue.svg)](https://python.org)
[![Django](https://img.shields.io/badge/Django-5.2-092E20.svg)](https://djangoproject.com)
[![PostgreSQL](https://img.shields.io/badge/PostgreSQL-17-336791.svg)](https://postgresql.org)

Веб-платформа для молодёжного центра «База», предназначенная для поддержки молодёжных инициатив, проведения мероприятий и развития ИТ-коммьюнити. Система автоматизирует процессы подачи заявок, информирования о событиях и управления контентом центра [citation:1][citation:3].

## О проекте

Платформа решает проблему децентрализации цифровых сервисов молодёжного центра и предоставляет единое пространство для взаимодействия участников и организаторов.

**Ключевые возможности:**
- 📅 Публикация и просмотр мероприятий и новостей
- 👥 Управление клубами по интересам
- 💡 Подача заявок на молодёжные инициативы
- 👤 Личный кабинет с историей заявок
- 🔐 Разграничение прав доступа (гость / пользователь / администратор)
- ⚙️ Административная панель для управления контентом и модерации заявок

## Технологический стек

### Бэкенд
- **Python 3.10+** — язык программирования
- **Django 4.2** — веб-фреймворк
- **Django ORM** — объектно-реляционное отображение
- **PostgreSQL 15** — реляционная база данных
- **Django Admin** — встроенная панель администратора

### Фронтенд
- **HTML5** — структура страниц
- **CSS3** — стилизация и адаптивная верстка
- **JavaScript** — интерактивные элементы
- **AJAX** — асинхронное обновление контента
- **Bootstrap** — адаптивная сетка

### Безопасность
- CSRF-токены — защита от межсайтовых атак
- Валидация форм на стороне клиента и сервера
- Защита от SQL-инъекций (встроенная в Django ORM)
